# 项目内容说明

## 项目定位
本项目是一个基于 **Tauri 2 + SvelteKit + TypeScript** 的移动端优先业务应用，当前包含两条主线功能：

1. 销售管理（客户、产品、销售单、送货单、报价单、数据统计）
2. 视频去水印工具（抖音/快手/小红书/TikTok 链接解析与下载）

数据默认保存在本地（`localStorage`），适用于离线场景。

## 技术栈
- 前端：SvelteKit、TypeScript、Tailwind CSS
- 桌面/移动容器：Tauri 2
- 后端能力（Tauri Rust 命令）：视频解析、视频下载、数据获取

## 当前功能范围（以代码为准）
### 业务管理
- 销售单管理：列表、搜索、排序、详情、编辑、删除、新建
- 客户管理：列表、搜索、排序、详情、编辑、删除、新建
- 产品管理：列表、搜索、排序、详情、编辑、删除、新建
- 报价单管理：创建、查看、编辑、复制、删除
- 送货单查看：按销售单生成送货单视图
- 数据看板：销售、欠款、客户排行、产品排行
- 标签与规格维护：全局标签/规格的增删改

### 工具功能
- 视频去水印页面：`/mobile/video-tools/tools`
- 支持平台：抖音、快手、小红书、TikTok
- 支持内容类型：视频、图文（图片）
- 支持能力：链接识别、解析、预览、下载保存

## 主要路由
- 首页：`/mobile`
- 销售管理：`/mobile/sales-management/sales`
- 客户管理：`/mobile/sales-management/customers`
- 产品管理：`/mobile/sales-management/products`
- 报价单管理：`/mobile/sales-management/quotation`
- 数据统计：`/mobile/sales-management/data`
- 视频去水印：`/mobile/video-tools/tools`
- 客服页：`/mobile/service`
- 我的页：`/mobile/profile`

## 数据存储
核心业务数据通过 `src/lib/utils/storage.ts` 统一管理，主要包含：
- `invoice_history`（销售单）
- `customers`（客户）
- `products`（产品）
- `quotations`（报价单）
- `quotation_products`（报价产品库）

## 运行与构建
### 开发
```bash
npm install
npm run dev
```

### 桌面构建
```bash
npm run tauri:build
```

### Android 构建
```bash
./build-android.sh
```
