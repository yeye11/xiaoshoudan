# 销售单生成器使用指南

## 功能概述

销售单生成器是一个基于 SvelteKit + Tauri 的桌面应用，可以帮助您快速创建专业的销售单据。

## 主要功能

### 1. 销售单创建
- **公司信息管理**：自动保存公司信息，下次使用时自动填充
- **客户信息录入**：支持客户名称、地址、电话等信息
- **商品明细管理**：动态添加、删除、复制商品项目
- **自动计算**：实时计算商品金额和总金额

### 2. 商品明细功能
- **动态添加**：点击"添加商品"按钮或使用快捷键 `Ctrl + Enter`
- **快速删除**：点击删除按钮或使用快捷键 `Ctrl + Delete`
- **项目复制**：一键复制相似商品项目
- **项目排序**：上移/下移调整商品顺序
- **多种单位**：支持件、个、套、米、平方米等多种计量单位

### 3. 预览和打印
- **实时预览**：查看格式化的销售单
- **打印功能**：支持直接打印销售单
- **复制功能**：将销售单内容复制到剪贴板
- **PDF导出**：导出为PDF文件（使用浏览器打印功能）

### 4. 历史记录管理
- **自动保存**：每次预览时自动保存到历史记录
- **搜索功能**：按客户名称、单据编号或制单人搜索
- **排序功能**：按日期、客户名称或金额排序
- **快速加载**：从历史记录快速加载之前的销售单
- **数据导出**：导出历史记录为JSON文件
- **批量管理**：清空所有历史记录

## 使用步骤

### 1. 设置公司信息
1. 在"公司信息"区域填写：
   - 公司名称
   - 公司地址
   - 联系电话
2. 信息会自动保存，下次使用时自动填充

### 2. 录入客户信息
1. 填写客户名称（必填）
2. 填写制单人（必填）
3. 选择送货日期
4. 可选填写客户地址和电话

### 3. 添加商品明细
1. 点击"添加商品"按钮添加新行
2. 填写商品信息：
   - 产品名称
   - 规格型号
   - 选择单位
   - 输入数量和单价
3. 系统自动计算金额和总计

### 4. 预览和打印
1. 点击"预览销售单"按钮
2. 检查销售单格式和内容
3. 使用操作按钮：
   - **返回编辑**：回到编辑界面
   - **复制**：复制销售单内容
   - **导出PDF**：使用浏览器打印功能
   - **打印**：直接打印

## 快捷键

- `Ctrl + Enter`：在商品明细表中添加新行
- `Ctrl + Delete`：在商品明细表中删除当前行

## 数据存储

- **本地存储**：所有数据保存在浏览器本地存储中
- **公司信息**：自动保存，持久化存储
- **历史记录**：最多保存50条记录
- **数据安全**：数据仅存储在本地，不会上传到服务器

## 技术特性

- **响应式设计**：支持不同屏幕尺寸
- **实时计算**：输入数据时实时更新金额
- **数据验证**：确保必填字段完整
- **用户友好**：直观的界面和操作流程

## 故障排除

### 常见问题

1. **数据丢失**
   - 检查浏览器是否清除了本地存储
   - 确保使用相同的浏览器和用户配置文件

2. **打印问题**
   - 确保浏览器允许打印
   - 检查打印机设置和纸张尺寸

3. **界面显示问题**
   - 刷新页面
   - 检查浏览器兼容性
